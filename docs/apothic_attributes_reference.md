# Apothic-Attributes 属性参考

## Apothic-Attributes 属性

| 属性名称 | 效果 |
|---------|------|
| `attributeslib:armor_pierce` | 平摊减少目标护甲值 |
| `attributeslib:armor_shred` | 百分比减少目标护甲值 |
| `attributeslib:arrow_damage` | 箭矢伤害乘数 |
| `attributeslib:arrow_velocity` | 箭矢速度乘数，伤害随速度成比例 |
| `attributeslib:cold_damage` | 近战附加冰霜魔法伤害，减速敌人 |
| `attributeslib:crit_chance` | 暴击触发几率 |
| `attributeslib:crit_damage` | 暴击伤害乘数，影响原版跳跃暴击 |
| `attributeslib:current_hp_damage` | 近战附加基于目标当前生命值的物理伤害 |
| `attributeslib:dodge_chance` | 闪避近战攻击或弹射物的几率 |
| `attributeslib:draw_speed` | 远程武器蓄力速度 |
| `attributeslib:experience_gained` | 破坏方块和击杀生物获得经验的乘数 |
| `attributeslib:fire_damage` | 近战附加火焰魔法伤害，点燃敌人 |
| `attributeslib:healing_received` | 所有治疗效果的乘数 |
| `attributeslib:life_steal` | 将造成伤害的百分比转化为自身治疗 |
| `attributeslib:mining_speed` | 挖掘速度乘数 |
| `attributeslib:overheal` | 将造成伤害的百分比转化为吸收护盾 |
| `attributeslib:prot_pierce` | 平摊减少目标保护点数 |
| `attributeslib:prot_shred` | 百分比减少目标保护点数 |
| `attributeslib:elytra_flight` | 启用鞘翅飞行能力 |
| `attributeslib:creative_flight` | 启用创造模式飞行能力 |

## Minecraft 原版属性

| 属性名称 | 效果 |
|---------|------|
| `minecraft:generic.max_health` | 最大生命值 |
| `minecraft:generic.knockback_resistance` | 击退抗性 |
| `minecraft:generic.movement_speed` | 移动速度 |
| `minecraft:generic.attack_damage` | 攻击伤害 |
| `minecraft:generic.attack_speed` | 攻击速度 |
| `minecraft:generic.attack_knockback` | 攻击击退 |
| `minecraft:generic.armor` | 护甲值 |
| `minecraft:generic.armor_toughness` | 护甲韧性 |
| `minecraft:generic.luck` | 幸运值 |
| `minecraft:generic.flying_speed` | 飞行速度 |

## Forge 扩展属性

| 属性名称 | 效果 |
|---------|------|
| `forge:entity_reach` | 与实体交互的距离（格） |
| `forge:block_reach` | 与方块交互的距离（格） |
