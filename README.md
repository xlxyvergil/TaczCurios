# TaczCurios Mod

一个为Minecraft 1.20.1版本设计的饰品模组，添加了多种具有特殊效果的饰品。

## 依赖项

- Curios API
- TaczAttributeAdd (TAA)

## 饰品列表与效果

### 传说级饰品 (EPIC)

#### 天火劫灭
- **描述**：踏上前来
- **效果**：以当前生命值的100%作为倍率提升通用伤害、爆炸范围和爆炸伤害，造成伤害后扣除100%当前生命值。这是一个高风险高回报的饰品，适合在关键时刻使用。
- **获取命令**：`/give @p tcc:heaven_fire_apocalypse`

#### 抵近射击Prime
- **描述**：近距离战斗的极致强化
- **效果**：增加165%特定枪械伤害属性（乘算），影响7种枪械类型：手枪、步枪、霰弹枪、狙击枪、冲锋枪、轻机枪、发射器。适合需要在近距离爆发输出的场合。
- **获取命令**：`/give @p tcc:close_combat_prime`

#### 撕裂Prime
- **描述**：极致的射击速度配合强大的穿透能力
- **效果**：增加55%射速(乘算)，增加2.2穿透(加算)。可以快速射击并穿透多个目标，适合对付集群敌人。
- **获取命令**：`/give @p tcc:ripping_prime`

#### 卑劣加速
- **描述**：以降低通用伤害为代价换取极致的射击速度
- **效果**：提升90%射击速度，降低15%通用伤害和全部特定枪械伤害。适合需要极高射速但可以接受较低伤害的场合。
- **获取命令**：`/give @p tcc:despicable_acceleration`

#### 我小心海也绝非鳝类
- **描述**：我小心海绝非鳝类，发射器力量的三倍强化
- **效果**：提升300%发射器伤害加成，300%爆炸伤害加成，300%爆炸范围加成。专门强化发射器类武器的饰品，对爆炸类武器有极大提升。
- **获取命令**：`/give @p tcc:careful_heart`

#### 烈焰风暴Prime
- **描述**：极致的爆炸掌控能力，战场上的烈焰君王
- **效果**：增加66%爆炸范围(乘算)，增加66%爆炸伤害(乘算)。大幅提升爆炸类武器的威力和影响范围，适合AOE输出。
- **获取命令**：`/give @p tcc:blaze_storm_prime`

#### 膛室Prime
- **描述**：极致提升狙击枪伤害
- **效果**：针对性提升100%狙击枪伤害，无法与膛室同时装备。
- **获取命令**：`/give @p tcc:chamber_prime`

#### 压迫点Prime
- **描述**：极致增强近战攻击力的神秘饰品
- **效果**：提升165%近战伤害。大幅提升使用枪械进行近战攻击时的伤害。
- **获取命令**：`/give @p tcc:oppression_point_prime`

#### 剑风Prime
- **描述**：极致扩大近战攻击范围的神秘饰品
- **效果**：提升3近战距离。大幅增加使用枪械进行近战攻击时的攻击距离。
- **获取命令**：`/give @p tcc:sword_wind_prime`

## 饰品互斥规则

- 烈焰风暴 与 烈焰风暴Prime 不能同时装备
- 天火圣裁 与 天火劫灭 不能同时装备
- 膛线 与 并合膛线 不能同时装备
- 膛室 与 膛室Prime 不能同时装备

## 使用方法

1. 确保已安装Curios API和TaczAttributeAdd模组
2. 将本模组放入mods文件夹
3. 启动游戏

## 装备饰品

1. 打开玩家背包
2. 点击饰品图标打开Curios界面
3. 将饰品拖拽到"Tacz饰品栏"中

## 注意事项

- 部分饰品效果强大但有负面效果，请谨慎使用
- 某些饰品之间存在互斥关系，无法同时装备
- 饰品效果基于TACZ枪械模组设计，建议搭配使用
- 天火系列饰品在造成伤害后会扣除使用者的生命值
